# 💰 expense-tracker

> 一个 Claude Code Skill，用自然语言输入零散费用，自动整理成格式化的 Excel 报账表。

## 是什么

报账前整理账目是件繁琐的事——费用零散、格式不统一、分类容易漏。这个 Skill 让你用说话的方式录入费用，Claude 自动解析、分类、写入 Excel，支持分批追加，最终生成一份可直接提交的报账表。

分类体系对应高校/科研机构报账系统，覆盖**日常报销**、**三公报销**、**差旅报销**和**酬金系统**全部类目。

**适用场景**：差旅报销、项目费用汇总、劳务酬金整理、日常零散报账

---

## 功能

- **自然语言输入**：直接描述费用，无需填表
- **批量解析**：一次输入多条，用分号隔开即可
- **智能推断**：自动识别费用类型、发票状态、付款方式
- **酬金类自动免发票**：编审费、课酬、科研劳务费等天然无发票，自动填充，不询问
- **使用前向导**：首次运行询问报账名称，作为文件名和表格标题
- **跨会话追加**：文件已存在时直接追加，不覆盖历史数据
- **双 Sheet 输出**：明细表 + 汇总表（SUMIF 公式，实时更新）
- **精美格式**：交替行色、冻结表头、金额格式化、汇总金色高亮

## 效果预览

**输入：**
```
昨天打车50，没发票，对私；今天午饭招待客户320元，有发票，对公；支付了张老师课酬800元，对私
```

**Claude 解析后展示确认表：**

| # | 发生日期 | 金额 | 费用类型 | 事由 | 付款方式 | 发票状态 |
|---|----------|------|----------|------|----------|----------|
| 1 | 2026-04-17 | 50 | 市内交通费 | 打车 | 对私 | 无发票 |
| 2 | 2026-04-18 | 320 | 公务招待费 | 招待客户午饭 | 对公 | 有发票 |
| 3 | 2026-04-18 | 800 | 课酬 | 张老师授课 | 对私 | **无发票（自动）** |

确认后写入 Excel，报告：新增 3 条，当前共 3 条，总金额 ¥1,170.00。

## 安装

### 方式一：手动安装（全局）

```bash
# macOS / Linux
git clone https://github.com/<你的用户名>/expense-tracker.git
cp -r expense-tracker ~/.claude/skills/

# Windows (PowerShell)
git clone https://github.com/<你的用户名>/expense-tracker.git
Copy-Item -Recurse expense-tracker "$env:USERPROFILE\.claude\skills\"
```

### 方式二：项目级安装

将 `expense-tracker/` 文件夹复制到项目的 `.claude/skills/` 目录，仅对该项目生效。

## 使用方法

在 Claude Code（CLI、VS Code 扩展、JetBrains 扩展均可）中触发：

```
/expense-tracker
```

或直接提到"报账"、"账目整理"、"记账"等关键词，Claude 会自动识别并启动。

**首次使用**：向导询问本次报账名称和输出目录（默认 `~/expense-reports/`）

**后续使用**：文件已存在则跳过向导，直接追加

## Excel 输出结构

**明细 Sheet**

| 序号 | 录入时间 | 发生日期 | 金额（元） | 费用类型 | 事由/摘要 | 付款方式 | 发票状态 | 经办人 | 备注 |

**汇总 Sheet**（SUMIF 公式，追加数据后自动更新）

- 总金额（金色高亮）
- 对公合计 / 对私合计
- 有发票合计 / 无发票合计 / 待补合计
- 各费用类型分类合计
